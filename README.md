# lab8
Выполнены все 8 заданий:
8.1. Написан обработчик события нажатия на кнопку «Изменить»: текущая строка в списке заметок пользователя должна быть заменена содержимым окна редактирования.
8.2. Добавлена в ежедневник вкладка «Купить», обеспечена корректность работы с записями на ней.
8.3. Написан обработчик события нажатия на кнопку «Удалить»: текущая строка в списке заметок пользователя должна быть удалена. После удаления выделенной должна стать следующая строка. Если удаляется последняя не единственная в списке строка, выделенной должна стать строка, ставшая последней после удаления. Выделенная строка должна отобразиться в окне редактирования.
8.4. Написан обработчик события нажатия на кнопку «Очистить»: содержимое текущей вкладки ежедневника удаляется.
8.5. Модифицирована программа следующим образом: добавление записи из окна редактирования происходит при нажатии клавиши ввода.
8.6. Написан обработчик события нажатия на кнопку «Закрыть»: приложение закрывается, данные текущего дня ежедневника автоматически сохраняются в соответствующий файл.
8.7. Добавлен к приложению строку состояния (StatusStrip), в которой отображены текущие дата, время и день недели.
8.8. Написаны все необходимые обработчики исключительных ситуаций, чтобы приложение было устойчиво к ошибкам.
